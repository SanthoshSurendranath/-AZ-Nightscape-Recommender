## Executive Summary

This repository presents a data science project focused on improving late-night dining discovery across Arizona through Natural Language Processing (NLP), topic modelling, exploratory data analysis, and recommendation logic. The goal is to transform restaurant-related information into a structured, business-ready experience that supports better user discovery and more relevant suggestions.

The project is designed as a portfolio-ready, corporate-style analytics solution that combines insight generation with practical recommendation capabilities.

---

## Business Objective

The objective of this project is to support users in identifying relevant late-night dining options in Arizona by:

- Organizing unstructured or semi-structured restaurant information
- Extracting topics and patterns from text-based data
- Supporting recommendation use cases with a data-driven workflow
- Improving discoverability through insight-led analysis

---

## Project Scope

This project centers on three core analytics areas:

| Area | Description |
|---|---|
| Exploratory Data Analysis | Understand patterns, distributions, and structure in the available data |
| Topic Modelling | Extract themes and recurring concepts from text content |
| Recommendation System | Surface relevant dining options based on the project logic and available signals |

---

## Methodology

### 1. Data Exploration
- Review the available dataset and identify key fields
- Examine data quality, structure, and descriptive patterns
- Identify text, categorical, and analytical variables relevant to the project

### 2. Text Processing and NLP
- Clean and prepare text fields for analysis
- Apply NLP techniques to uncover meaningful themes
- Organize the content into insights that can support recommendation logic

### 3. Topic Modelling
- Group related terms and patterns into interpretable topics
- Use discovered themes to understand the dominant subjects in the dataset
- Translate topic patterns into business-friendly observations

### 4. Recommendation Layer
- Use the analytical output to support item discovery
- Present recommendations in a format suitable for user exploration
- Align recommendations with the late-night dining use case

### 5. Insight Presentation
- Summarize findings in a clear and structured manner
- Support portfolio and stakeholder review with concise storytelling
